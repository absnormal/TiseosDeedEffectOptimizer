## 夢幻模擬戰浮空城契約模擬器

[模擬器連結](https://absnormal.github.io/TiseosDeedEffectOptimizer/)

可以依照擁有的角色尋找契約最優解

在螢幕最上方選擇建築

在右邊點擊刪去"你沒有的角色"會變成灰色 :arrow_right: 從可選名單中去除

再次點擊會變回彩色 :arrow_right: 加回可選名單

左邊會顯示目前可選名單的最優解

正在選取的角色會有藍色外框

- 每次改動名單會重新計算一次最優解
- 各建築角色共通
- 計算時最低連結數為2(會算太久，可能會修改)
- 4個建築的角色可能會被共用
- 有時候會跑比較久，要等一下:smiling_face_with_tear:
